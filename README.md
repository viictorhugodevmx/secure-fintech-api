# Secure Fintech API

Séptimo proyecto del programa **Bankaool .NET Fintech Bridge Lab**.

API REST con ASP.NET Core Web API para practicar seguridad backend fintech.

## Objetivo

Construir una API protegida con:

- Usuarios
- Roles
- Login
- JWT
- Endpoints protegidos
- Autorización por rol

## Stack

- Ubuntu
- .NET 8
- C#
- ASP.NET Core Web API
- JWT
- Swagger/OpenAPI
- Postman

## Conceptos

| Concepto | Significado |
|---|---|
| Authentication | Confirmar quién eres |
| Authorization | Confirmar qué puedes hacer |
| JWT | Token firmado para autenticar requests |
| Role | Nivel de permiso del usuario |
| Protected endpoint | Endpoint que requiere token |

## Comandos

Restaurar:

```bash
dotnet restore

```
Build:

dotnet build

Ejecutar:

dotnet run

Modo watch:

dotnet watch run
